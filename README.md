# 🏃‍♀️ ACL Recovery Female Athletes — Resource Hub

A curated, evidence-based resource hub for sports medicine teams focused on **anterior cruciate ligament (ACL) injury recovery and rehabilitation in female athletes**.

This repository aggregates clinical trial data from ClinicalTrials.gov, structured rehabilitation protocol summaries, and trend analyses to support clinical decision-making and research.

---

## 📁 Repository Structure

| File | Description |
|------|-------------|
| [`trial-summaries.md`](./trial-summaries.md) | Detailed summaries of the 5 most relevant ACL clinical trials |
| [`rehabilitation-protocols.md`](./rehabilitation-protocols.md) | Structured rehab protocols extracted from trial interventions |
| [`trend-analysis.md`](./trend-analysis.md) | Statistical trend analysis: by phase, status, sponsor type, and country |
| [`data/trials-raw.json`](./data/trials-raw.json) | Raw structured trial metadata (NCT IDs, status, sponsors, interventions) |

---

## 🔬 Data Source

All trial data is sourced from **[ClinicalTrials.gov](https://clinicaltrials.gov)** — the U.S. National Library of Medicine's registry of clinical studies.

- **Search Query:** Anterior Cruciate Ligament + Rehabilitation
- **Population Focus:** Female athletes / women
- **Filters Applied:** RECRUITING, COMPLETED, ACTIVE_NOT_RECRUITING
- **Total Studies in Analysis:** 338
- **Data Retrieved:** June 2026

---

## 🩺 Key Trial Areas Covered

1. **Neurostimulation (tDCS)** — Brain stimulation to improve quadriceps recovery post-ACLR (NCT07128602)
2. **Plyometric + Hamstring Training** — Prevention protocol for female amateur athletes (NCT07047833)
3. **VR Mindfulness Meditation** — Addressing injury-related fear and brain activity post-ACLR (NCT05527171)
4. **Blood Flow Restriction (BFR) + Total Motion Release (TMR)** — Strength and balance restoration (NCT07589309)
5. **Oxandrolone Subdermal Implant** — Pharmacological adjuvant for muscle recovery post-ACLR (NCT06974526)

## 📊 At-a-Glance Trends (338 total studies)
